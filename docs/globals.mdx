---
title: 'Global Styles'
---

Sometimes you might want to insert global css like resets or font faces. You can use the `Global` component to do this. It accepts a `styles` prop which accepts the same values as the `css` prop except it inserts styles globally. Global styles are also removed when the styles change or when the Global component unmounts.

```jsx
// @live
import { Global, css } from '@emotion/react'

render(
  <div>
    <Global
      styles={css`
        .some-class {
          color: hotpink !important;
        }
      `}
    />
    <Global
      styles={{
        '.some-class': {
          fontSize: 50,
          textAlign: 'center'
        }
      }}
    />
    <div className="some-class">This is hotpink now!</div>
  </div>
)
```

#### Loading fonts with @font-face

If you wish to load custom fonts using the [@font-face API](https://developer.mozilla.org/en-US/docs/Web/API/FontFace), you can return an array from the `styles` prop of `<Global />`, splitting '@font-face' definitions in different objects.

```jsx
// @live
import { Global } from '@emotion/react'

render(
  <div>
    <Global
      styles={[
        {
          "@font-face": {
            fontFamily: "Lato",
            fontWeight: 400,
            src: 'url("https://cdnjs.cloudflare.com/ajax/libs/lato-font/3.0.0/fonts/lato-normal/lato-normal.woff") format("woff"), url("https://cdnjs.cloudflare.com/ajax/libs/lato-font/3.0.0/fonts/lato-normal/lato-normal.woff2") format("woff2")'
          }
        },
        {
          "@font-face": {
            fontFamily: "Lato",
            fontWeight: 700,
            src: 'url("https://cdnjs.cloudflare.com/ajax/libs/lato-font/3.0.0/fonts/lato-bold/lato-bold.woff") format("woff"), url("https://cdnjs.cloudflare.com/ajax/libs/lato-font/3.0.0/fonts/lato-bold/lato-bold.woff2") format("woff2")'
          }
        },
        {
          ".lato": {
            fontFamily: '"Lato", sans-serif',
          },
          ".normal": {
            fontWeight: 400,
          },
          ".bold": {
            fontWeight: 700,
          }
        }
      ]}
    />
    <div className="lato">
      <h1 className="bold">This uses Lato now</h1>
      <p className="normal">Both normal and bold variants are available</p>  
    </div>
  </div>
)
```
